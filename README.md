## 🚀 Sobre o Projeto

O **BookTrack API** é uma API REST construída com ASP.NET Core com o objetivo de praticar e demonstrar boas práticas de desenvolvimento backend.

O projeto utiliza **Entity Framework Core** para persistência de dados e segue o padrão **DDD (Domain-Driven Design)**, garantindo organização, escalabilidade e separação de responsabilidades.

## 🗄️ Banco de Dados

O projeto utiliza **SQL Server** com abordagem **Code First**.

Para criar o banco:

```bash
dotnet ef database update
```
# Clone o repositório
git clone https://github.com/Joao-V36/BookTrack-API.git

# Acesse a pasta
cd BookTrack-API

# Execute a aplicação
dotnet run
